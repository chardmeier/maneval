10000 test set samples used for manual evaluation, in which each sample contains 4 consecutive sentences as a group, divided by ' _eos ' symbol. The data is lowercased, tokenised text. 

source.txt: source-side sentences;

baseline.txt: the sentence-level NMT translations;

docrepair.txt: the repair translation of DocRepair model;

transference.txt: the repair translation of Transference model.
